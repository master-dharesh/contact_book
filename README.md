ContactBook
ContactBook is a simple Python based contact management application.
It is designed for beginners and is fully accessible for screen reader users such as NVDA.
Folder Name
ContactBook
Features
Add new contacts with name and phone number
View all saved contacts
Search contacts by name
Update existing contact details
Delete contacts
Keyboard friendly and accessible
Uses SQLite database (no internet required)
Technologies Used
Python 3
SQLite3 database
Command Line Interface
Project Structure
ContactBook folder contains:
contact_book.py – main Python program file
contacts.db – SQLite database file (created automatically)
README file – project information
How to Run the Project
Step 1: Install Python 3 on your computer
Step 2: Open Command Prompt or Terminal
Step 3: Go to the ContactBook folder
Step 4: Run the program using this command
python contact_book.py
How the Program Works
The program shows a numbered menu
You choose options using the keyboard
All contacts are saved in a local SQLite database
Contact names are stored and searched easily
Accessibility Notes
Works well with NVDA screen reader
Clear text messages and prompts
No mouse needed
Simple menu based navigation
Who Can Use This Project
Python beginners
Blind and visually impaired learners
Students learning database CRUD operations
Anyone practicing SQLite with Python
Future Improvements
Import and export contacts
